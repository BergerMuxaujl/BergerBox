Commands GIT

  • git clone path-to-repository
  • git status - статус текщего локального репозитория
  • git add path-to-file - добавить новый файл под контроль GIT
  • git commit -am "you comment"
  • git push - отправить все изменения в репозиторий на сервер
  • git pull - забрать свежие обновления
